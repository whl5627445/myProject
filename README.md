## 介绍：
  ### 这是一个基于OpenModelica项目后端omc的Web API项目
## 项目目标：
  ### 本项目的目标是为有需要的用户提供一个基于omc的Web API，方便用户在不安装OMEdit的情况下使用omc的功能
## 安装：
  ```markdown
  omc官方的docker镜像，最低版本1.21
  Golang最低版本1.22
  此项目并无前端组件，所以需要使用者自行开发图形渲染代码
  启动项目非常简单，设置好config里的各项配置后进行编译，例如go build main.go
  直接执行可执行文件即可
